## Implementation Plan
1. Backend (Motoko):
   - Admin account management: auto-generate initial credentials, allow password change
   - Movie CRUD: id, title, posterUrl, videoUrl, createdAt
   - PIN code management: generate random 6-char alphanumeric code, store with expiry (20min/30day/90day), device token binding, auto-expiry check
   - Auth session: admin session management
   - Queries: getMovies (paginated), getMovie, listPinCodes (admin), validatePinCode (returns device token), checkDeviceToken

2. Frontend:
   - PinEntry page: 6-char input, submit, error state (Монгол)
   - Home page: movie grid 4x5, pagination, site logo header, Settings icon top-right
   - Movie detail page: full-screen embedded video from Dropbox URL
   - Admin panel: login form, dashboard with tabs for Movies and PIN Codes management
   - Device token stored in localStorage; checked on every page load
